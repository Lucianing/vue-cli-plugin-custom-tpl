# vue-cli-plugin-custom-tpl

[Vue CLI 3.0](https://github.com/vuejs/vue-cli) 自定义模板插件

## 安装

使用vue-cli 3.0 常规创建一个项目
```
vue create my-app
```

```
cd my-app
vue add custom-tpl
```
